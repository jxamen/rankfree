# 03. 키워드 분석 화면 스펙 (UI 참고)

> 태훈님 제공 참고 UI(2026-07-10). 키워드 트렌드 분석(B1) 화면의 목표 구성.
> 데이터 소스: 네이버 검색광고 keywordstool(웹세션) + 크롤링 보강 (분석 진행 중 — [research-naver-ads-session.md] 참조).
> 스타일: Cal.com 디자인 시스템(흰 카드, 도넛/라인차트, 라이트그레이 배경).

## 상단 요약 바
- 키워드 최초 등장일 · 키워드 등급(D 등) · 상업 키워드 여부(예: 비상업 키워드) · 검색광고 포함 여부

## 지표 카드 (4열 그리드)
1. **월간 검색량** — PC / Mobile / Total (예: 37,800 / 91,200 / 129,000)
2. **월간 콘텐츠 발행량** — 블로그 / 카페 / 전체 (예: 81,900 / 13,200 / 95,000)
3. **N월 예상 검색량** — 최근 검색량 / 월말까지 예상 검색량 (증감률)
4. **콘텐츠 포화지수** — 블로그 / 카페 / 전체 (%, 포화도)

## AI 인사이트
- "이 키워드, 실제로 공략할 가치가 있을까요?" — AI 분석 카드(1회 무료 생성, 크레딧 차감). 검색·경쟁·콘텐츠 상황 요약.

## 연관 키워드 테이블 (핵심)
- 전체 N개(예 200개), 탭: 전체/강남/맛집/강남맛집/추천/강남맛집…(키워드 그룹)
- 컬럼: 키워드 · 월간 검색량(Total) · 블로그 누적 발행량 · 절사 유사도(높음/중간)
- **무료는 상위 10개, 나머지는 유료(회원 전용)** — 전체 리스트/고급 필터/CSV 다운로드는 유료. (요금: 쇼핑처럼 무료 횟수 후 유료 / 구독)

## 검색량 트렌드
- 라인차트(월별). 필터: 전체 / 연령·성별별(10~50대 남녀). 기간 선택, 키워드 비교, CSV 다운로드.

## 분포 차트
- **월별 검색 비율**(막대) · **요일별 검색 비율**(막대)
- **연령별 검색 비율**(막대) · **성별 검색 비율**(도넛)
- **이슈성**(도넛, 이슈성%) · **정보성/상업성**(도넛)

## 인기글 TOP 7
- 순위 · 구분 · 제목/링크 · 발행일 등. **회원 전용(유료)** — 상위 몇 개만 무료.

## 액션 배치 순서
- PC / Mobile 각각 네이버 통합검색 영역 배치 순서(플레이스/파워링크/네이버클립/블로그/인기글/웹사이트 등)

## 구현 메모
- 무료/유료 경계: 상위 요약·연관키워드 상위 10개·기본 트렌드는 무료, 전체 리스트·인기글·CSV·AI인사이트는 유료(구독/횟수).
- 차트는 SVG 자체 렌더 또는 경량 라이브러리. Cal.com 토큰(모노크롬 + 액센트 절제).
- 데이터 확보 가능 범위는 keywordstool 실측 분석 결과에 맞춰 확정.
