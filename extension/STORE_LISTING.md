# 크롬 웹스토어 등록 텍스트

> 크롬 웹스토어 개발자 대시보드 → 스토어 등록정보의 **설명(상세 설명)** 란에 아래 구분선 밑 본문을 그대로 붙여넣는다.
> **요약(Summary)** 은 [manifest.json](manifest.json) 의 `description`(132자 제한)에서 자동으로 가져오므로 요약 변경 시 새 패키지 업로드가 필요하다.
> 섹션·메뉴 구성은 rankfree 콘솔 메뉴와 1:1 대응(플레이스·쇼핑·마케팅·블로그·셀프마케팅·API).
> 최종 수정: 2026-07-22

랭크프리(RankFree) — 네이버 마케팅을 위한 무료 분석 도구

네이버 쇼핑·플레이스·키워드·블로그·상품 리뷰를 한 곳에서 분석합니다.
사장님·마케터·셀러가 내 순위와 시장을 빠르게 파악하도록 돕습니다.

■ 플레이스
· 순위 추적 — 키워드별 네이버 플레이스 순위를 매일 자동 기록하고 순위 추이·리뷰 변화를 추적합니다.
· 경쟁 분석 — 상위 노출 업체와 내 플레이스의 리뷰·저장·SEO 신호를 나란히 비교합니다.
· 매장 분석(플레이스 개별 분석) — 내 매장 1곳을 키워드 기준으로 진단해 순위와 키워드 유사도·관련성·랭킹 점수, 리뷰 신호를 확인합니다.
· 스마트플레이스 리포트 — 방문·유입·리뷰·예약 통계를 모아 한 장의 리포트로 보여줍니다.

■ 쇼핑
· 쇼핑 순위 추적 — 상품·업체 × 키워드 조합의 네이버 쇼핑 검색 순위를 매일 추적합니다.
· 시장 분석 — 키워드 시장 규모·매출 추정·상위 점유율·경쟁 구도를 분석해 진입 전략을 잡도록 돕습니다.
· 셀러력 — 내 상품의 SEO·노출 지수를 경쟁 상품과 항목별로 비교 진단합니다.
· 상품 분석 — 상품 리뷰를 모아 감정분석·재구매율·옵션별 판매 비중·약점 키워드를 분석합니다.

■ 마케팅 (키워드)
· 키워드 분석 — 월 검색량·경쟁 강도, 성별·연령 분포, 12개월 검색 트렌드, 연관 키워드·함께 많이 찾는 키워드를 한 번에 봅니다.
· 키워드 추천 — 시드 키워드 하나로 추천 키워드를 뽑고 검색량 기준으로 쓸 만한 키워드만 골라냅니다.
· 키워드 대량 분석 — 여러 키워드를 한 번에 등록해 검색량·경쟁 지표를 일괄 분석합니다.
· 키워드 인사이트 — 카테고리별 키워드 허브에서 검색량·트렌드와 AI 인사이트 문서를 열람합니다.

■ 블로그
· 블로그 수집 — 키워드로 검색에 노출된 블로그를 수집하고 방문·이웃·전문성 지수를 분석합니다.
· 블로그 지수 분석 — 특정 블로그의 지수·활동·영향력을 점수로 확인합니다.
· 저장 블로거 — 분석하며 저장한 블로거를 키워드별로 모아 보고 엑셀로 내려받습니다(체험단 후보 관리에 유용).

■ 셀프마케팅
· 체험단·블로그 리뷰·SNS·참여형 리워드 등 마케팅 상품을 콘솔에서 직접 주문하고 진행 현황을 관리합니다.

■ API
· 순위 추적·경쟁 분석·키워드 분석 데이터를 REST API로 제공합니다. 콘솔에서 API 키를 발급하고 rankfree.kr/developers 문서를 확인하세요.

■ 이렇게 동작합니다
네이버 쇼핑/검색/상품/플레이스 페이지에서 확장을 열면, 그 페이지를 기반으로 분석 결과를 보여줍니다. 사용자가 요청할 때만 필요한 데이터를 수집해 요약하며, 일반 브라우징 기록은 수집하지 않습니다.

■ 이용 안내
· rankfree.kr 무료 회원 가입/로그인이 필요합니다.
· 순위·점수·시장 규모 등은 자체 추정값이며, 네이버 공식 수치가 아닙니다.
